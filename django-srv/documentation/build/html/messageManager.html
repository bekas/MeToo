CHAPTER 4 Message manager¶

Система управления пакетами и сообщениями¶

Одна из основных систем сервера.Выполняет задачу обработки запросов от клиента: в зависимости от типа входящего пакета формирует соответствующий пакет - ответ. Выполняет данную задачу для запросов от любых клиентов. Использует системы управления пользователеми, событиями, подписками на событиями, систему управления сессиями.
